# specd

A specification-driven development CLI tool.

## Install

### macOS / Linux

```sh
curl -sSL https://stackific.com/specd/install.sh | sh
```

### Windows (PowerShell)

```powershell
irm https://stackific.com/specd/install.ps1 | iex
```

### npm

```sh
npm install -g @stackific/specd-cli
```

### pipx

```sh
pipx install specd
```

## Uninstall

### macOS / Linux

```sh
curl -sSL https://stackific.com/specd/uninstall.sh | sh
```

### Windows (PowerShell)

```powershell
irm https://stackific.com/specd/uninstall.ps1 | iex
```

### npm
